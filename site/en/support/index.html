Technical issues: GitHub Issues

Location is not working

Open MouBCN → You → Settings → Location permission. If you previously denied access, open system settings and allow location while using the app.

Delete saved data

You can remove Home or Work individually from You. In Settings → Privacy and data, you can delete all places, favourites, history, and the last selection. Language and appearance are kept.

Incorrect or missing information

Schedules, arrivals, and alerts depend on TMB, FGC, and TRAM Barcelona. Include the operator, line, stop, and approximate time when reporting a problem.

Before contacting support

  • Check your internet connection.
  • Update MouBCN to the latest available version.
  • Close and reopen the app.
